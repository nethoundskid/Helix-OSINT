# HELIX OSINT v2.0

> A fast, lightweight, and modular OSINT (Open-Source Intelligence) reconnaissance toolkit built with Python.

HELIX is designed to collect publicly available information from multiple online sources through a clean interactive interface or command-line mode. It is intended for security researchers, penetration testers, bug bounty hunters, and anyone performing legitimate OSINT investigations.


---

## Available Modules

| Module         | Description                               |
| -------------- | ----------------------------------------- |
| GitHub         | GitHub profile reconnaissance             |
| Username Hunt  | Search usernames across multiple websites |
| Discord Server | Discord invite information                |
| Discord User   | Discord User ID lookup                    |
| IP Geolocation | IP information                            |
| Domain Recon   | DNS, headers, Wayback, subdomains         |
| NPM            | Package information                       |
| PyPI           | Python package information                |
| Phone Analyzer | International phone lookup                |
| Telegram       | Username and channel lookup               |
| Email OSINT    | Email reconnaissance                      |
| Reddit         | User activity                             |
| Hacker News    | User profile                              |
| Keybase        | Identity proofs                           |
| Dev.to         | Developer profile                         |
